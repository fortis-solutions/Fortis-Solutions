## 🔎 Threat Hunting & Detection Project

- **[Threat Hunt – Unauthorized TOR Usage Detection](https://github.com/fortis-solutions/SK-CYBERLAB/blob/main/labs/threat-hunting-sentinel-unauthorized-tor-usage)**  
  Conducted a threat hunt using **Microsoft Sentinel and Defender Advanced Hunting (KQL)** to detect unauthorized TOR browser installation and usage.  
  The investigation included:

  - File artifact discovery using **DeviceFileEvents**
  - Process execution analysis using **DeviceProcessEvents**
  - Network activity analysis using **DeviceNetworkEvents**
  - Evidence documentation with **MITRE ATT&CK mapping and NIST-style incident reporting**

  The lab demonstrates how security teams can detect anonymization tools used to bypass monitoring and network security controls.
